WordPress の RSS フィードに投稿のアイキャッチ画像（サムネイル）を追加するシンプルなプラグインです。Snow Monkey Blocks の RSS ブロックにも対応しています。

== Description ==
このプラグインは、WordPress の RSS フィードに投稿のアイキャッチ画像（サムネイル）を `<media:thumbnail>` タグとして追加します。Snow Monkey Blocks の RSS ブロックでもサムネイル画像が表示されるようになります。

- 1ファイル構成で外部依存はありません。
- WordPress Coding Standards に準拠しています。

== Installation ==
1. プラグインファイル（add-thumbnail-to-rss-feed.php）を `wp-content/plugins` フォルダにアップロードします。
2. 管理画面「プラグイン」から「Add Thumbnail to RSS Feed」を有効化します。

== Frequently Asked Questions ==
= サムネイルが表示されない場合は？ =
投稿にアイキャッチ画像が設定されているか確認してください。
